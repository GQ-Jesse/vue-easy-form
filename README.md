# vue-easy-form

## Project setup
```
npm install
```
### Compiles and hot-reloads and mock data for development
```
npm run mock
```

### Compiles and hot-reloads for development
```
npm run serve
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
